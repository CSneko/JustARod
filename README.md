# 只是根棍子
> `JustARod` by @CSNeko

这是[More_end_rod](https://github.com/CSneko/More_end_rod/)模组的续作，但是需要toNeko作为前置，你可以将其理解为toNeko的附属模组喵

你需要在 `Minecraft 1.21.1` 使用 `Fabric` 加载器运行，并且需要安装 [Fabric API](https://modrinth.com/mod/fabric-api) 和 [Fabric Language Kotlin](https://modrinth.com/mod/fabric-language-kotlin) 以及 [toNeko](https://modrinth.com/mod/tonekomod) 作为前置哦

喜欢这个模组的话记得把star点上(｢・ω・)｢

可以来[爱发电](https://afdian.com/a/cccry)支持作者获取整合包哦
